Guía de Despliegue RADIX Phase 0

Sigue esta lista para asegurar que todos los componentes se suban correctamente a tu hosting y la aplicación funcione al 100%.

📂 Carpeta: /radix_api/ (Núcleo)

  • config.php Credenciales de DB. ¡Revisar antes de subir! CONFIG
  • db_setup.php Ejecuta esto primero para crear las tablas. SETUP
  • matrix_logic.php El cerebro de los Beneficios y el Salto de $100. CORE
  • clon_logic.php Activación automática del Agente IA. CORE
  • user_data.php API que alimenta el Dashboard del Usuario. API
  • admin_global_stats.php API que alimenta tu Dashboard de Dueña. API
  • registro.php Sistema de registro de nuevos líderes. CORE

📄 Dashboards (Raíz del Sitio)

  • dashboard.php Tu nuevo Panel Premium para los usuarios. PANEL
  • admin_dashboard.php Tu Panel de Control como Dueña. MASTER
  • index.html Página de login/landing principal. WEB

Pasos para el Lanzamiento:

Paso 1: Configurar Base de Datos

Crea la base de datos en tu hosting (CPanel/Plesk) y actualiza **config.php** con el nuevo nombre, usuario y contraseña.

Paso 2: Subir y Activar

Sube todos los archivos por FTP. Luego, abre en el navegador el archivo **tudominio.com/radix_api/db_setup.php** para que se creen las tablas automáticamente.

Paso 3: Primer Registro

Regístrate tú misma primero para ser la ID #1 (Cuenta Maestra) y que todos los que vengan después queden debajo de ti.
